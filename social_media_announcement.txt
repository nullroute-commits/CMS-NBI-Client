# CMS-NBI-Client v2.0.0 Released - Modern Async Python Client for Calix Management System

Hey r/Python! Just released v2.0.0 of CMS-NBI-Client, a complete modernization of the Python client for Calix Management System NBI.

**Major improvements:**
- Full async/await support with aiohttp
- Docker images reduced from ~1GB to ~150MB using Alpine
- Comprehensive test suite and documentation
- Circuit breakers and retry logic
- Type hints throughout

**Tech stack:**
- aiohttp for async HTTP
- Pydantic for configuration
- structlog for structured logging
- pytest for testing
- MkDocs for documentation

GitHub: https://github.com/nullroute-commits/CMS-NBI-Client
Docs: https://nullroute-commits.github.io/CMS-NBI-Client/
